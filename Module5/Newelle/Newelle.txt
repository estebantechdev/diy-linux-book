Install as flatpak from flathub

$ sudo flatpak install flathub io.github.qwersyk.Newelle

Install permissions:

$ sudo flatpak install flatseal

Run:

$ flatpak run io.github.qwersyk.Newelle

Install AIs (LLMs) using API keys (OpenAI's, Gemini, Grok, etc.)

Setup microphone and language (Gnome).

Setup language:

Voice to text > Google > en-US or es-ES
