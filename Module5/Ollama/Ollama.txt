Download Ollama:

Visit https://ollama.com/

Choose Linux installation.

Install with one command:
$ curl -fsSL https://ollama.com/install.sh | sh

The installation will:
Download Ollama
Install ollama to /usr/local/bin
Create a symlink from /etc/systemd/system/default.target.wants/ollama.service to /etc/systemd/system/ollama.service.

The Ollama API is now available at 127.0.0.1:11434.

If no Nvidia GPU is detected, Ollama will let you know:
WARNING: No NVIDIA/AMD GPU detected. Ollama will run in CPU-only mode.

You can also follow this link for detailed GPU installation instructions: https://github.com/ollama/ollama/blob/main/docs/linux.md

Install GPU Drivers. You can list the Nvidia GPU like this:

$ nvidia-smi

References:
Video card.txt

Verify that Ollama is working.

The first time you run this command will install the chosen model, in this case Phi3 mini 128k instruct (2.2 GiB).

$ ollama run phi3:mini-128k

You can use prompts like this or use Ollama system message to modify the model behavior with "custom instructions."

The default language for the Phi3 SLM model is always English, and it cannot be configured to another language by default. If you need the model to understand and respond in another language, you should instruct it explicitly before starting. For example, you can say: "Let's speak French."

List /set commands:

>>> /set

Set a system message:

Use this:
>>> /set system You are very precise when answering questions. Don't talk about yourself, what you can do for me, or even how you're going to help, unless you're asked to.

Alternatively, you can create a Modelfile with a "custom instructions" prompt.

Save the model name using commbase as prefix:

>>> /save commbase-phi3-mini.

You have saved it for later use.

Exit the CLI:

>>> /bye

Open the new model name:

$ ollama run commbase-phi3-mini

Or from another model, use:

>>> /load commbase-phi3-mini

Loading model 'commbase-phi3-mini'

Show the current model information:

>>> /show info

This will show the next information:

Model details:
Family              phi3
Parameter Size      3.8B
Quantization Level  Q4_K_M

List your models list:

$ ollama list

Show more options:

$ ollama --help

Start and stop the Ollama service:

$ systemctl start ollama.service

$ systemctl stop ollama.service

Ollama has a REST API for running and managing models.

Developers can communicate with Ollama models using Python, JavaScript, or LangChain libraries.
